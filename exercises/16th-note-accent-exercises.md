# 16th Note Accent Exercises

## Practice Guide

These exercises focus on developing control and precision with 16th note patterns and accent placement. Each exercise uses a 4-note grouping annotation system where `1` represents an accented note and `0` represents an unaccented note.

### Notation System

- `1` = Accented note
- `0` = Unaccented note
- Groups are separated by spaces (e.g., `1000 1100 0111 0110`)
- Each bar contains 16 sixteenth notes (4 groups of 4)

### Practice Variations

#### 1. **Straight Single** (RLRL...)

Play all notes with alternating right and left hand without dynamics variation. Use this as a foundation to internalize the accent pattern without technique changes.

#### 2. **Single Hand Lead**

- **Right hand lead**: Accents are played by the right hand, unaccented by left hand
- **Left hand lead**: Accents are played by the left hand, unaccented by right hand
- Develops independence and control with each hand focusing on specific notes

#### 3. **Fill Unaccented Notes with Diddles**

Replace all unaccented notes (0s) with double strokes (diddles). This adds texture and challenges coordination while maintaining the accent structure.

#### 4. **Play Accents as 32nd Notes**

Play all accented notes (1s) as 32nd notes while keeping unaccented notes (0s) as 16th notes. This creates rhythmic variation and develops speed control.

#### 5. **Play Accents with Herta**

Replace accented notes (1s) with Herta stickings (RLr or LRl), which creates a triplet feel displacement while maintaining the accent placement concept.

#### 6. **Play Accents with Flam**

Add a grace note (flam) before each accented notes, enhancing the accent clarity and developing flam consistency.

#### 7. **Linear Pattern - Accent on Hand, Non-accent on Foot**

Play accented notes (1s) with hand strokes and unaccented notes (0s) with foot pedal. This develops linear independence and coordination between hands and feet, creating a continuous flowing pattern without hand-based dynamics variation.

#### 8. **Doubled Note Value (32nd Notes)**

Practice these exercises with 32nd notes instead of 16th notes—doubling your note subdivision while keeping the accent pattern identical. This is an excellent technique for speed development at manageable metronome tempos.

## Practice Tips

1. **Start Slowly**: Begin each exercise at a comfortable tempo where you can execute cleanly
2. **Focus on Evenness**: Maintain consistent stick height and spacing between sixteenth notes
3. **Accent Clarity**: Ensure accents are clearly distinct from unaccented notes through volume or articulation
4. **Variation Progression**: Practice each variation in order for maximum benefit before combining them
5. **Repetition**: Repeat exercises multiple times, gradually increasing tempo
6. **Listen Actively**: Pay attention to rhythm consistency and accent placement throughout
